# Glossary

This file defines canonical domain terminology for AI agents.

Use these terms consistently across documentation, code, APIs, and UI labels.

Do not invent synonyms or redefine terminology unless explicitly requested.

## PCF

Product Carbon Footprint.

제품 단위 탄소 배출량.

Unit:
kgCO₂e

---

## Activity Data

탄소 배출 활동 데이터.

Examples:

- electricity usage
- material usage
- transport distance

---

## Emission Factor

활동량 기준 탄소 배출 계수.

Formula:

Emission = Activity Data × Emission Factor

---

## Emission

계산된 탄소 배출량.
Unit: kgCO₂e

---

## GHG Scope

탄소 배출 책임 범위.
